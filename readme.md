## Node js, Express, socket.io chat app

- Javascript
- socket.io v.4
- Express
- Node js
- Responsive design
- Chat options: group messaging, private messaging, zoom-in images (desktop version)
- Chat messages' variations: text, emojis, text + emojis, pictures, pdf
